# Raitha-Varta 🌾

Raitha-Varta is an Android agriculture app that provides farmers with short flash-card style crop advisories instead of long PDFs. The app offers daily farming tips, crop guidance, weather-based suggestions, and multilingual support to help farmers make better decisions easily.

## Features

* Daily farming tips
* Crop-wise advisories
* Kannada, English & Hindi support
* Offline access using Room Database
* Notifications and expert suggestions

## Tech Stack

* Kotlin
* XML
* Firebase
* Room Database
* Android Studio

## Run the App

1. Open project in Android Studio
2. Sync Gradle
3. Run on Emulator or Android device
